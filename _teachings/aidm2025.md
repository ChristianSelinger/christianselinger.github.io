## Course Overview

This course provides a comprehensive introduction to disease transmission modeling and practices. Students will:

- Build compartmental models
- Calculate critical thresholds
- Perform numerical simulations
- Assess model parameter inference from epidemiological data

## Prerequisites

- Basic programming knowledge in R
- Linear algebra

- Modeling Infectious Disease by Matt Keeling and Pejman Rohani
